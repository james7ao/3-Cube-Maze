# @cube-maze/ui

Planned shared React component library (HUD, Controls, Toast, Help panel, etc.).

**Status: stub only.** This package currently has a `package.json` but no `src/` directory. The real components live in [apps/web/src/components/](../../apps/web/src/components/) (`HUD.jsx`, `Controls.jsx`, `Toast.jsx`, `Help.jsx`, `RotationHint.jsx`). Moving them here (and updating `apps/web` to import from `@cube-maze/ui`) is future work — do not assume this package exports anything usable yet.
